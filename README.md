# GDLWebcamp

Aplicacion web de una conferencia sobre desarrollo web a llevarse a cabo en la Ciudad de Guadalajara - México. En el front-end se muestra los eventos, invitados, 
precios de pases, ubicación, fotografías, etc. finalmente el pago por la compra de los pases es a travéz de PayPal. Y en el back-end se tiene un panel de administración
para poder alimentar el front-end.
Este proyecto lo realizé en el curso de **Desarrollo Web Completo con HTML5, CSS3, JS AJAX PHP y MySQL** de manos de mi instructor Juan Pablo De la torre Valdez. El dicho curso se encuentra alojado en la plataforma [Udemy](https://www.udemy.com/course/desarrollo-web-completo-con-html5-css3-js-php-y-mysql/).
